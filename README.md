# LitLens

Инструмент для методологического обзора: статьи на диске, единый словарь терминов, подсветка на сайте и в локальном reader.

## Стек

| Часть | Технология |
|-------|------------|
| Хранилище | Папка `~/LiteratureReview` (JSON + HTML + текст) |
| Мост | **Node.js 18+** + **Express** на `http://127.0.0.1:17321` |
| Reader | Статический HTML/JS (отдаётся сервером) |
| Подсветка | Общий движок `shared/highlight.js` |
| Браузер | **Chrome / Edge** расширение (Manifest V3) |

Загрузка статей **по URL из reader не используется** — после логина сохраняйте страницу через расширение или вставляйте текст/HTML.

## Быстрый старт

```bash
cd ~/litlens
npm install
npm start
```

Откройте reader: http://127.0.0.1:17321/

### Расширение Chrome

1. `chrome://extensions` → Режим разработчика → **Загрузить распакованное**
2. Папка: `~/litlens/extension`
3. На странице статьи (после логина) → иконка LitLens → **Сохранить эту страницу на диск**
4. Правый клик по выделенному слову → **LitLens: добавить как термин** (в первую категорию)

## Структура данных

```
~/LiteratureReview/
  terms.json
  articles/
    a1b2c3d4/
      meta.json    # title, url, addedAt, notes
      page.html    # снимок контента
      text.txt     # plain text для поиска
```

Переменная окружения `LITLENS_DATA` — другой путь к данным.

## Workflow

1. Запустить `npm start` (можно добавить в автозагрузку).
2. Читать статью на сайте издателя → **Сохранить** в расширении.
3. Термины задавать в reader или через контекстное меню на странице.
4. В reader переключаться между статьями — те же цвета подсветки.
5. Вкладка **Поиск** — вхождения по всем `text.txt`.
